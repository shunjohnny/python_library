🗂 Pathlib Modern Guide

Pythonのモダンなファイルパス操作ライブラリ

📍 パス操作の基本

パスの作成と基本的な情報取得。クロスプラットフォームで安全なパス操作が可能です。

from pathlib import Path # パスの作成 path = Path('docs/reports/monthly/report.txt') # 基本的な情報取得 path.parent # docs/reports/monthly path.parents[0] # monthly path.parents[1] # reports path.name # report.txt path.suffix # .txt path.stem # report path.parts # ('docs', 'reports', 'monthly', 'report.txt')
parent: 直近の親ディレクトリ parents: 親ディレクトリのリスト name: ファイル名 suffix: 拡張子 stem: 拡張子なしファイル名 parts: パスの構成要素

💫 ファイル操作とチェック

ファイルの読み書きと存在確認。テキストとバイナリ両方のモードに対応しています。

# ファイルの存在確認と種類チェック path.exists() # ファイルが存在するか path.is_file() # ファイルであるか path.is_dir() # ディレクトリであるか path.is_symlink() # シンボリックリンクか # テキストファイル操作 path.write_text('Hello, Pathlib!', encoding='utf-8') content = path.read_text(encoding='utf-8') # バイナリファイル操作 path.write_bytes(binary_data) data = path.read_bytes() # ファイル情報 stats = path.stat() size = path.stat().st_size # ファイルサイズ
exists: 存在確認 is_file: ファイル判定 is_dir: ディレクトリ判定 stat: ファイル情報

📁 ディレクトリ操作

ディレクトリの作成、削除、内容の列挙などの高度な操作が可能です。

# ディレクトリ作成 path.mkdir( parents=True, # 親ディレクトリも作成 exist_ok=True # 既存でもエラーにしない ) # ディレクトリ内容の列挙 for item in path.iterdir(): print(item) # すべてのファイルとディレクトリ # パターンマッチング python_files = path.glob('*.py') # 直下のPythonファイル all_files = path.rglob('*.*') # 再帰的にすべてのファイル docs = path.glob('**/*.txt') # すべてのテキストファイル # パス変更 new_path = path.rename(path.with_name('new_name.txt')) moved_path = path.replace(other_path) # 既存ファイルも上書き
mkdir: ディレクトリ作成 iterdir: コンテンツ列挙 glob: パターンマッチ rename: 名前変更 replace: 移動/上書き

🔧 パス操作ユーティリティ

パスの変更、結合、解決などの便利な操作が提供されています。

# パスの結合 new_path = Path('docs') / 'reports' / 'monthly' # ホームディレクトリからの相対パス home = Path.home() / 'documents' # カレントディレクトリ操作 current = Path.cwd() relative = path.relative_to(current) # パス文字列の変更 modified = path.with_suffix('.md') # 拡張子変更 renamed = path.with_name('other.txt') # ファイル名変更 absolute = path.absolute() # 絶対パスを取得 resolved = path.resolve() # シンボリックリンクを解決
home: ホームディレクトリ cwd: カレントディレクトリ with_suffix: 拡張子変更 with_name: 名前変更 absolute: 絶対パス取得 resolve: パス解決
